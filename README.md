# Apache-Search

## Tech Stack
- React
- React Router Dom
- Express
- MongoDB
- Material-UI

## Instructions to run locally
```
git clone https://github.com/ahmedrzakhan/Apache-Search.git
cd Apache-Search
cd front-end
yarn install
yarn start
cd back-end
yarn install
yarn start
```
